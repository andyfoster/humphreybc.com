The first blog post.

So this is my first blog post. I'm attempting to get a simple blog up and running using the existing site bits and pieces I have with Node.js.

Right now, I have to type my blog post out by hand directly into this HTML file which is a wee bit shit. Ideally, I'd like to have the ability to:

  • Write my posts in markdown
  • Store them in some /articles folder or an articles repository
  • Pull them in and convert them to HTML
  • Add them to my blog.html file
  • Have some basic meta information like a date attached to each
  • Eventually have a unique URL for each blog entry for sharing, etc

So that's that. I have looked into using Wheat, which is what howtonode.org uses. However I can't work out how it actually does things - I cloned the repo and poked around and all I could find was a server.js file which has a little bit of stuff that isn't super helpful.

I did manage to make a simple theme thing for the blog which you can see on the top right. That's kind of nifty.

That's all for now - maybe one day I'll have a proper blog here with proper content! Who knows.
